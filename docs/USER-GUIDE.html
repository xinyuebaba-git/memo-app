    1. 快速开始

    1.1 访问应用

    在线访问： 

    http://localhost:5173

    部署后访问： 

    https://your-domain.vercel.app

    1.2 首次使用

  • 打开浏览器访问应用
  • 看到欢迎页面和空列表
  • 点击「+ 新建」创建第一个备忘录
  • 输入标题和内容
  • 点击「保存」

    • 1.3 系统要求

    浏览器：

  • Chrome 90+
  • Safari 14+
  • Firefox 88+
  • Edge 90+

    • 设备：

  • 桌面电脑（Windows/Mac/Linux）
  • 平板电脑（iPad/Android）
  • 智能手机（iPhone/Android）

    3. 使用指南

    3.1 创建备忘录

    方法 1：点击新建按钮

  • 点击右上角「+ 新建」按钮
  • 弹出编辑窗口
  • 输入标题（必填）
  • 输入内容（可选）
  • 添加标签（可选）
    • - 直接输入标签名称 - 多个标签用逗号分隔 - 例如：工作，重要，会议
  • 点击「保存」

    • 方法 2：从空状态创建

  • 如果没有备忘录，显示空状态
  • 点击「创建第一个备忘录」
  • 后续步骤同上

    • 技巧：

  • 标题尽量简洁明了
  • 标签有助于后续查找
  • 可以只写标题，内容后续补充

    • ---

    3.2 编辑备忘录

    步骤：

  • 点击备忘录卡片（任意位置）
    • - 看到 ✏️ 图标提示 - 卡片边框变紫色

  • 编辑窗口打开
    • - 自动填充当前内容 - 可修改标题、内容、标签

  • 修改内容
    • - 标题：直接修改 - 内容：编辑文本 - 标签：添加或删除

  • 保存修改
    • - 点击「保存」按钮 - 或按 Ctrl/Cmd + S

  • 取消编辑
    • - 点击「取消」按钮 - 或点击右上角 × - 或点击窗口外部

    提示：

  • 编辑不会自动保存，需手动点击保存
  • 取消编辑会丢弃所有修改
  • 标题不能为空

    • ---

    3.3 删除备忘录

    步骤：

  • 找到要删除的备忘录

  • 点击卡片右上角的 🗑️ 图标
    • - 注意：不要点击卡片其他区域（那是编辑）

  • 弹出确认对话框
    • - 显示备忘录标题 - 询问是否确认删除

  • 点击「删除」确认
    • - 或点击「取消」放弃

  • 删除完成
    • - 备忘录从列表移除 - 无法恢复（重要！）

    警告：

  • ⚠️ 删除操作不可逆
  • ⚠️ 删除后数据无法恢复
  • ⚠️ 删除前请确认

    • ---

    3.4 使用标签

    添加标签：

  • 创建或编辑备忘录
  • 在标签输入框输入
  • 多个标签用逗号分隔
  • 点击保存

    • 标签格式： 

    工作
生活
学习
工作，重要，会议
项目 A，紧急

    查看标签：

  • 每个备忘录显示其标签
  • 标签以胶囊形状显示
  • 显示 🏷️ emoji

    • 按标签筛选：

  • 点击顶部的标签按钮
    • - 「全部」- 显示所有备忘录 - 「工作」- 只显示工作相关 - 「生活」- 只显示生活相关 - 等等...

  • 列表自动筛选
    • - 只显示包含该标签的备忘录 - 其他标签隐藏

  • 取消筛选
    • - 点击「全部」 - 或点击其他标签

    管理标签：

  • 标签自动创建，无需预先定义
  • 删除所有包含某标签的备忘录后，该标签自动消失
  • 标签名称可以随时修改

    • ---

    3.5 搜索备忘录

    步骤：

  • 在搜索框输入关键词

  • 实时搜索结果
    • - 标题包含关键词 - 内容包含关键词 - 不区分大小写

  • 清除搜索
    • - 删除搜索框内容 - 或按 Esc 键

    搜索示例： 

    会议      → 搜索包含"会议"的备忘录
工作      → 搜索包含"工作"的备忘录
2026      → 搜索包含"2026"的备忘录

    组合使用：

  • 可以先按标签筛选，再搜索
  • 也可以先搜索，再按标签筛选

    手机/平板优化：

    4. 快捷键

    桌面端快捷键：

    快捷键 | 功能 | 说明
    --------|------|------
    Ctrl/Cmd + N | 新建备忘录 | 快速创建

    5. 常见问题

    Q1: 数据会同步到云端吗？

    A: 不会。当前版本使用 LocalStorage，数据仅保存在您的本地浏览器中。

    优点：

  • ✅ 数据安全，不会泄露
  • ✅ 无需登录，开箱即用
  • ✅ 无需网络，离线可用

    • 缺点：

  • ❌ 无法跨设备同步
  • ❌ 清除浏览器数据会丢失
  • ❌ 不同浏览器数据独立

    • 解决方案：

  • 定期导出数据备份
  • 使用同一浏览器访问
  • 未来版本会支持云端同步

    • ---

    Q2: 最多能创建多少备忘录？

    A: LocalStorage 限制约 5-10MB，可存储数千个备忘录。

    实际测试：

  • 纯文本备忘录：约 5000-10000 个
  • 含少量标签：约 3000-5000 个
  • 建议定期清理无用备忘录

    • ---

    Q3: 如何备份数据？

    方法 1：浏览器开发者工具

  • 按 F12 打开开发者工具
  • 进入 Application/存储 标签
  • 找到 LocalStorage
  • 找到 memo_app_memos 键
  • 复制值（JSON 字符串）
  • 保存为 .json 文件

    • 方法 2：导出功能（待实现）

    未来版本会添加一键导出功能。

    ---

    Q4: 删除的备忘录能恢复吗？

    A: 不能。删除操作不可逆，请谨慎操作。

    建议：

  • 删除前确认
  • 重要内容先备份
  • 使用标签标记重要备忘录

    • ---

    Q5: 支持富文本编辑吗？

    A: 当前版本仅支持纯文本。

    未来计划：

  • 支持 Markdown
  • 支持富文本编辑器
  • 支持图片上传

    • ---

    Q6: 可以导出为 PDF 吗？

    A: 当前版本不支持。

    临时方案：

  • 打开备忘录
  • 使用浏览器打印功能（Ctrl/Cmd + P）
  • 选择「另存为 PDF」

    • ---

    Q7: 支持多语言吗？

    A: 当前版本仅支持中文。

    未来计划：

  • 英文
  • 日文
  • 其他语言

    • ---

    Q8: 数据丢失了怎么办？

    可能原因：

  • 清除了浏览器缓存
  • 使用了隐私模式
  • 更换了浏览器
  • LocalStorage 被清空

    • 预防措施：

  • 定期备份数据
  • 不要使用隐私模式
  • 固定使用同一浏览器
  • 导出重要备忘录

    • 恢复方法：

  • 如果有备份，导入备份文件
  • 否则无法恢复

    • ---

    6. 数据管理

    6.1 数据存储位置

    浏览器： LocalStorage 键名： memo_app_memos 格式： JSON 数组

    6.2 查看存储数据

    Chrome/Edge：

  • 按 F12 打开开发者工具
  • 点击 Application 标签
  • 展开 LocalStorage
  • 点击您的域名
  • 找到 memo_app_memos

    • Safari：

  • 按 Cmd + Option + I
  • 点击 Storage 标签
  • 展开 LocalStorage
  • 找到 memo_app_memos

    • Firefox：

  • 按 F12 打开开发者工具
  • 点击 Storage 标签
  • 展开 LocalStorage
  • 找到 memo_app_memos

    • 6.3 导出数据

    步骤：

  • 打开开发者工具
  • 找到 memo_app_memos
  • 复制值（JSON 字符串）
  • 保存为 .json 文件
  • 妥善保存备份文件

    • 示例数据： 

    [
  {
    "id": "memo_1708776000000_abc123",
    "title": "工作会议记录",
    "content": "讨论 Q2 目标...",
    "tags": ["工作", "会议"],
    "createdAt": 1708776000000,
    "updatedAt": 1708776000000
  }
]

    6.4 导入数据

    步骤：

  • 打开开发者工具
  • 找到 memo_app_memos
  • 双击值
  • 粘贴 JSON 数据
  • 按 Enter 保存
  • 刷新页面

    • 警告：

  • ⚠️ 导入会覆盖现有数据
  • ⚠️ 导入前请备份当前数据
  • ⚠️ 确保 JSON 格式正确

    • 6.5 清理数据

    清理单个备忘录：

  • 使用删除功能

    • 清理所有数据：

  • 打开开发者工具
  • 找到 memo_app_memos
  • 右键 → 删除
  • 刷新页面

    • 清理浏览器数据：

  • 在浏览器设置中清除网站数据
