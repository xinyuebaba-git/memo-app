# 📝 备忘录系统

一个轻量级、跨平台的备忘录管理应用，支持桌面端和移动端使用。

**版本：** 1.1.0  
**创建时间：** 2026-02-24  
**最后更新：** 2026-02-26（安全升级）  
**开发团队：** OpenClaw 软件开发 Agent 团队

---

## ✨ 功能特性

- 📱 **跨平台支持** - 桌面端和移动端完美适配
- 📝 **多备忘录管理** - 创建、编辑、删除多个备忘录
- 🏷️ **标签分组** - 自定义标签，快速筛选和分组
- 🔍 **搜索功能** - 快速查找备忘录
- 💾 **本地存储** - 数据保存在本地，安全隐私
- ⚡ **快速响应** - 流畅的用户体验
- 🔐 **密码哈希** - bcrypt 加密存储，防止密码泄露
- 🔒 **数据加密** - AES 加密备忘录内容，保护隐私

## 📖 使用说明

### 创建备忘录

1. 点击首页右上角「+ 新建」按钮
2. 输入标题和内容
3. 添加标签（可选）
4. 点击「保存」

### 编辑备忘录

1. 在列表中点击要编辑的备忘录
2. 修改内容
3. 点击「保存」

### 删除备忘录

1. 在备忘录卡片上点击 🗑️ 图标
2. 确认删除

### 使用标签

1. 创建/编辑时输入标签名称
2. 多个标签用逗号分隔
3. 点击顶部标签按钮筛选

## 📝 API 文档

### useMemos Composable

```typescript
const {
  memos,          // 备忘录列表
  createMemo,     // 创建备忘录
  updateMemo,     // 更新备忘录
  deleteMemo,     // 删除备忘录

  getAllTags,     // 获取所有标签
```

## 🐛 常见问题

### Q: 数据会同步到云端吗？
A: 当前版本使用 LocalStorage，数据仅保存在本地。如需多设备同步，可部署后端服务。

### Q: 最多能创建多少备忘录？
A: LocalStorage 限制约 5-10MB，可存储数千个备忘录。

### Q: 如何备份数据？
A: 浏览器 LocalStorage 数据，可通过浏览器开发者工具导出。

### Q: 支持导出备忘录吗？
A: 当前版本不支持，后续版本会添加导出为 TXT/Markdown 功能。

### Q: 密码和备忘录内容是否加密？🔐
A: **是的！** v1.1.0 起已添加加密保护：
- 密码使用 bcrypt 哈希存储（不可逆）
- 备忘录内容使用 AES 加密存储
- 详见 [SECURITY.md](./SECURITY.md)

### Q: 如何启用备忘录加密？
A: 在浏览器控制台运行 `enableEncryption()` 即可启用。加密后，即使查看 LocalStorage 也只能看到密文。
